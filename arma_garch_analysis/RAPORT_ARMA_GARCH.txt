1. CEL I DANE
================================================================================

Cel: Dopasowanie optymalnego modelu ARMA-GARCH do dziennych stóp zwrotu S&P 500
     uwzględniającego efekt dźwigni i grube ogony rozkładu.

Dane: S&P 500 ETF (SPY), okres: 2017-01-04 do 2018-12-31, N = 501 obserwacji

Modele testowane:
  • Średnia warunkowa: ARMA(p,q), wybór według AIC
  • Zmienność warunkowa: GARCH(1,1), EGARCH(1,1), GJR-GARCH(1,1)
  • Rozkłady: Normal, Student's t
  • Razem: 9 kombinacji (3 modele × 3 rozkłady)

2.3. Parametry najlepszego modelu
--------------------------------------------------------------------------------
GJR-GARCH: σ²ₜ = ω + α·ε²ₜ₋₁ + γ·ε²ₜ₋₁·I(εₜ₋₁<0) + β·σ²ₜ₋₁

┌──────────┬──────────┬───────────┬──────────┬───────────────────────────┐
│ Parametr │ Wartość  │  Std.Err  │ p-value  │       Interpretacja       │
├──────────┼──────────┼───────────┼──────────┼───────────────────────────┤
│ ω        │  0.0178  │  0.0086   │  0.038*  │ Wariancja bazowa          │
│ α        │  0.0295  │  0.0415   │  0.478   │ Symetryczny szok          │
│ γ        │  0.2391  │  0.1187   │  0.044*  │ EFEKT DŹWIGNI             │
│ β        │  0.8355  │  0.0515   │ <0.001***│ Persistencja zmienności   │
│ ν        │  3.998   │  0.843    │ <0.001***│ Stopnie swobody (rozkład t)│
└──────────┴──────────┴───────────┴──────────┴───────────────────────────┘

Kluczowe wnioski:
  • γ = 0.24 > 0 (p < 0.05): Negatywne szoki zwiększają zmienność
    (α+γ)/α = 9.1× bardziej niż pozytywne (efekt dźwigni)
  • α + β + γ/2 = 0.985: Wysoka persistencja (half-life ≈ 46 dni)
  • ν ≈ 4: Bardzo ciężkie ogony (P(|r|>4σ) jest 40× częstsze niż w N(0,1))

4. NAJWAŻNIEJSZE WNIOSKI
================================================================================

1. MODEL OPTYMALNY: GJR-GARCH(1,1) + Student's t + Okno 200

   Uzasadnienie:
     ✓ Najniższe AIC (913.23) spośród 9 modeli
     ✓ Uwzględnia efekt dźwigni (γ=0.24, p<0.05)
     ✓ Grube ogony (ν=4.0)
     ✓ Pokrycie CI 94.92% ≈ 95% teoretyczne
     ✓ Stabilny w małych i dużych próbach

2. EFEKT DŹWIGNI jest statystycznie istotny i ekonomicznie znaczący:

   γ = 0.24 (p = 0.044) → negatywne szoki zwiększają zmienność 9× bardziej

   Implikacja: Value-at-Risk dla pozycji długich powinien być asymetryczny

3. ROZKŁAD NORMALNY jest NIEADEKWATNY:

   Niedoszacowanie pokrycia:  -6.4pp dla N=100,  -1.2pp dla N=200
   Przyczyna: Nie uwzględnia grubych ogonów (kurtoza = 6.4 vs 3.0)

4. OKNO 200 OBSERWACJI jest optymalne:

   Kompromis: Wystarczająco długie dla estymacji + krótkie dla adaptacji
   Praktyka: ≈ 1 rok tradingowy (banki używają 1-2 lat)

5. NARUSZENIA CI są ASYMETRYCZNE (74% poniżej):

   Przyczyny:
     • Efekt dźwigni (szybsza reakcja na spadki)
     • Panika rynkowa (Q4 2018: korekta -20%)
     • Clustering zmienności w kryzysach

   Wniosek: Szczególna ostrożność w zarządzaniu ryzykiem spadkowym


5. REKOMENDACJE PRAKTYCZNE
================================================================================

Dla zarządzania ryzykiem:
  ✓ Zastąp GARCH(Normal) przez GJR-GARCH(Student's t) w modelach VaR
  ✓ Okno estymacyjne: 200-250 obserwacji (≈1 rok)
  ✓ Asymetryczne limity: downside VaR wymaga bufora +20-30%
  ✓ Reestymacja: kwartalna (nie częściej!)

Dla zarządzania portfelem:
  ✓ Po wzroście zmienności: obniż ekspozycję na 4-6 tygodni (half-life=46 dni)
  ✓ Hedging: więcej zabezpieczeń w trendzie spadkowym (efekt asymetrii)
  ✓ Sygnał alertu: σ̂ₜ₊₁ > 2% → rozważ redukcję ekspozycji


6. OGRANICZENIA I DALSZE BADANIA
================================================================================

Ograniczenia:
  • Krótki okres (2017-2018) - brak pełnego cyklu koniunkturalnego
  • Jedno aktywo (S&P 500) - brak dywersyfikacji
  • Brak zmiennych egzogenicznych (stopy, inflacja, sentiment)

Kierunki rozszerzeń:
  • Okres 2008-2024 (kryzysy: subprime, COVID, inflacja)
  • Multivariate DCC-GARCH dla portfela aktywów
  • Markov-switching GARCH dla zmian reżimu
  • Trading strategies z dynamiczną zmiennością
